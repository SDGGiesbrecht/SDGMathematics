Subtractable

public protocol Subtractable: Addable

A type that can do scalar subtraction.

Note

Subtractable is distinct from Negatable to allow whole number types to perform subtraction.

Note

Unlike SignedNumber, Subtractable types do not need to conform to Comparable, allowing conformance by two‐dimensional vectors, etc.
